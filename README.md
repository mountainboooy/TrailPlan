# ジョン・ミューア・トレイルとは
ジョン・ミューア・トレイル（John Muir Trail）は、アメリカの長距離自然歩道。カリフォルニア州内を、ヨセミテ峡谷（ヨセミテ国立公園）からマウント・ホイットニーまで、340キロメートルにわたって縦走する。トレイルの大部分はパシフィック・クレスト・トレイルの一部になっている。アメリカにおける「自然保護の父」と呼ばれるナチュラリスト、ジョン・ミューアにちなんで命名されている。
